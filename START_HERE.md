# 🚀 Welcome to the Sample Work!

## Quick Start for Candidates

### 1. Installation
```bash
npm install
```

### 2. Read the Task
Open `README.md` and read the complete task description.

### 3. Start the Project
```bash
npm start
```

You should see:
```
🚀 Starting Jira Stale Issues Reporter...
Mock data loaded: 15 issues
✅ Report completed successfully
```

### 4. Start Implementation
Open `src/index.ts` and start implementing your solution.

## Important Files

| File | Description |
|------|-------------|
| `README.md` | 📋 Complete task description |
| `src/index.ts` | 💻 Where you'll implement your solution |
| `src/types/index.ts` | 📝 TypeScript type definitions |
| `src/mockData/jiraIssues.ts` | 🎭 Test data (15 mock issues) |
| `examples/` | 💡 Example code for inspiration |

## Need Help?

- **Setup issues?** Make sure Node.js v16+ is installed
- **TypeScript errors?** Check if `npm install` completed successfully
- **Questions about the task?** Contact your point of contact

## Timeline
- Setup & understanding: 30 min
- Core implementation: 1.5 hours
- Error handling: 30 min

## Good luck! 🎯
